About the Game

This game is called Whack-a-Mole.  
The main idea is click the red mole when it comes on the box to get points.  
You have only 30 seconds to click as many moles as you can.  
The mole appears in random boxes.  

---

How to Play

1. Open the folder Tamilselvan.B_151.  
2. Open index.html in browser (chrome is better).  
3. Click Start Game to begin.  
4. Click the red mole to increase your score.  
5. Game ends when timer become 0.  
6. Click Play Again to restart.
